GanttChart

This page shows how to create GanttChart using FlexChart.

Getting Started

Steps for getting started with the FlexChart to customize a GanttChart in Angular2 applications:

  1. Add references to Angular2, Wijmo, and Wijmo's Angular2 directives.
  2. Add a component to provide data and logic.
  3. Add a FlexChart to the page, set chartType to Bar and bind the itemsSource to the data.
  4. Add a FlexChartSeries to FlexChart and set its binding property to the names of the fields that contain the start and end dates for each task.
  5. Customize AxisY to hide major grid and show minor grid.

Dependencies & Percent Complete

Customize the chart by setting its itemFormatter property to a function that adds elements to show task dependencies and percent complete.
