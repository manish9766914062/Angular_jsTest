Highlight invalid cells

This sample shows how to highlight invalid cells of a FlexGrid in the exported document.

For the FlexGrid's invalid cells to be highlighted in the exported document the following steps should be performed:

  1. Make sure that FlexGrid.showErrors property is turned on.
  2. Specify the errorCellStyle property of the styles export setting. This style will be applied to each invalid cell being rendered.

Note: there is no need to specify the errorCellStyle property if custom cell content mode is used.

In this sample the following code highlights invalid cells with the pale red color: 

wijmo.grid.pdf.FlexGridPdfConverter.export(flexGrid, 'FlexGrid.pdf', {
   styles: {
        errorCellStyle: {
            backgroundColor: 'rgba(255, 0, 0, 0.3)'
        }
    }
})
