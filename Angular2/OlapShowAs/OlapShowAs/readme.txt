OlapShowAs (Angular 2)
-------------------------------------------------------------------------------------
Show all settings for the PivotField.showAs property.

This app creates a simple pivot table showing the effect of all available settings 
for the PivotField.showAs property.

It also allows you to open an Excel file containing the same data and a pivot
table configured with the same settings so you can compare the results.

System requirements
====================
Please refer to the description in the readme.txt file of the Wijmo Explorer for Angular 2 sample.
