HistoryOfFlexGrid (Angular 2)
------------------------------------------------------------------------------
The long and storied history of the FlexGrid Control, from 1996 to 2015.

This sample uses CSS to make the FlexGrid look like its older versions running 
on older operating systems, even as far back as Windows 95!

System requirements
====================
Please refer to the description in the readme.txt file of the Wijmo Explorer for Angular 2 sample.
