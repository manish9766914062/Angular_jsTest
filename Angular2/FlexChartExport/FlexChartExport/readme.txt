FlexChartExport (Angular 2)
------------------------------------------------------------------------------
Shows how to export a FlexChart control to an image.

The sample shows how to export the Wijmo FlexChart control to PNG, JPG, and BMP using saveImageToFile function.

CanvasRenderEngine file is an extension of FlexChart. 
It provides the ability to render FlexChart using Canvas instead of SVG and supports export FlexChart in IE9+.

System requirements
====================
Please refer to the description in the readme.txt file of the Wijmo Explorer for Angular 2 sample.
