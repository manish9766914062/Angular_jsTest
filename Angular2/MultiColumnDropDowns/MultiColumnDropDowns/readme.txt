MultiColumnDropDowns (Angular 2)
-------------------------------------------------------------------------------------------------------------
Shows how you can use CSS to create multi-column dropdowns for several controls.

The sample uses the "dropDownCssClass" property to show how you can use CSS to create multi-column dropdowns
for controls including the ComboBox, MultiSelect, and FlexGrid.

The sample also uses the same CSS to create a multi-column ListBox control.

System requirements
====================
Please refer to the description in the readme.txt file of the Wijmo Explorer for Angular 2 sample.
